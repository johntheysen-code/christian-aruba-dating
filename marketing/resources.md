## 🧭 How to bookmark in 60 seconds

**Chrome / Edge (desktop):**
1. Press `Ctrl + Shift + B` to show the bookmarks bar
2. Right-click the bar → **Add folder** → name it **Amor y Fe**
3. Visit each URL above → press `Ctrl + D` → choose the Amor y Fe folder → Save

**Safari (Mac):**
1. View → Show Bookmarks Bar
2. Bookmarks → Add Bookmark Folder → name it Amor y Fe
3. Visit each URL → Bookmarks → Add Bookmark → choose the Amor y Fe folder

**Safari (iPhone):**
1. Tap Bookmarks icon at bottom → Edit → New Folder → Amor y Fe
2. Visit each URL → tap Share → Add Bookmark → choose the Amor y Fe folder
